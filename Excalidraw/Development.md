- Fast Startup: Containers start in milliseconds instead of minutes.

üèõÔ∏è The Architecture of Docker

Docker consists of several key components: ^3exGAHGr

1. Docker Engine: The main service that runs containers.
2. Docker Daemon (dockerd): The background process managing containers.
3. Docker CLI (docker): The command-line interface to interact with Docker.
4. Docker Images & Containers:
    > Image: A blueprint of an application (like a VM snapshot).
    > Container: A running instance of an image.
5. Docker Hub: A cloud registry to store and share Docker images. ^gt3xRZtx
